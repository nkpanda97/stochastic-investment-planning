39   ## Deterministic model formulation (MOD1)<br> 
40   <center>$ 
41   \min_{x\geq0}{\left\{\underbrace{\sum_{i=1}^nc_ix_i}_{\text{investment cost}}+\underbrace{\sum_{i=1}^n\sum_{j=1}^kq_iT_jy_{ij}}_{\text{operating cost}}\right\}}$</center><br> 
42   <ul><ul><ul><ul><em>subject to:</em><br></ul></ul></ul></ul> 
43   <center> 
44   $ 
45   \begin{align} 
46   &\sum_{i=1}^n x_i\geq \sum_{j=1}^k \xi_j^{max}\\ 
47   &\sum_{i=1}^nc_ix_i\leq c_{max}\\ 
48   &\sum_{j=1}^ky_{ij}\leq x_i\quad \quad i=\{1,\cdots,n\}\\ 
49   &\sum_{i=1}^ny_{ij}\geq \xi_j \quad \quad j=\{1,\cdots,k\} 
50   \end{align} 
51   $ 
52   </center> 
53   #%% md 
54   ## Expected value (EV) solution (MOD1) 
55   For expected value solution, the expectation of each random parameters is used as input to the deterministic model as shown below:<br> 
56    
57   Solve MOD1 with:<br> 
58   <ul> <ul> <ul> <ul> $ 
59   \begin{align} 
60   &\xi_j = \bar{\xi_j}&&\quad\quad \forall \: j={1,2,3}\\ 
61   \text{where;}\\ 
62   &\bar{\xi_j}=\mathbb{E}[\xi_j]=\sum_{l=1}^3\xi_j^{(l)}\mathcal{P}(\xi_j^{(l)}) &&\quad\quad\forall \: j={1,2,3} 
63   \end{align} 
64   $</ul> </ul> </ul> </ul><br><br> 
65   Expected Value (EV): 
66   <ul> <ul> <ul> <ul> 
67   $ 
68   EV:=\min_{x\geq0}\left(g(E[\xi], x)\right)=g(\bar{\xi}, \bar{x})\\ 
69   \bar{x}\text{ is the optimal solution of MOD1 by using }\xi = \bar{\xi} 
70   $</ul> </ul> </ul> </ul> 
71   #%% md 
72   ## Expected value of using the EV solution (EEV) (MOD1.1) 
73   The expected value of using the EV solution can be calculated using the following equation:<br> 
74    
75   \begin{align} 
76   EEV:= \mathbb{E}_{\xi}\left[g(\bar{x}, \xi)\right] 
77   \end{align} 
78   EEV can be exactly calculated for m random discrete variables at d values each by solving $d^n$ linear programs. The EEV is calculated using sample mean approximation as follows:<br> 
79    
80   - For $i = 1,...,L$ where $L=d^m$ and for this particular problem $d=m=3$: 
81       - Sample $\xi_i$ from the discrete realizations<br> 
82       - Compute the optimal value of $x_i$ for $\bar{\xi}$: $\bar{x}_i$<br> 
83       - $EEV_l = \sum_{i=1}^n c_i\bar{x}_i+Q(\bar{x}, \xi^{(l)})$ where, $l$ is the $l^{th}$ scenario. Here $l={1,2,...,27}$ 
84       - Finally, $EEV = \frac{1}{L}\sum_{l=1}^LEEV_l$ 
85       - $Q(\bar{x}, \xi^{(l)})$ can be written as a linear program as follows: 
86   \begin{align} 
87       \min_{y\geq0}\left\{\sum_{i=1}^n \sum_{j=1}^kq_iT_jy_{ij}\right\} 
88   \end{align} 
89    
90   \begin{align} 
91   \text{Subject to:}&\\ 
92   &\sum_{j=1}^ky_{ij}\leq \bar{x}_i\quad \quad i=\{1,\cdots,n\}\\ 
93   &\sum_{i=1}^ny_{ij}\geq \xi_j^{(l)} \quad \quad j=\{1,\cdots,k\} 
94   \end{align} 
95   #%% md 
96   ## Wait-and-See problem 
97   The wait-and-see problem is formulated by solving MOD1 for all possible realizations of $\xi_j$. As $\xi_j$ is discrete and mutually independent there are a total of 27 realizations.<br> 
98   The mean value of wait-and-see solution is denoted by WS and can be formulated as:<br><br> 
99   <center> 
100  $ 
101  WS:=\mathbb{E}_{\xi}\left[min_{x\geq0}\:g(\xi, x)\right]=\mathbb{E}_{\xi}\left[g(\xi,\hat{x}(\xi))\right] 
102  $</center>

105  ## Stochastic recourse model 
106  The stochastic 2-stage model can be formulated by chainging the objective function of the deterministic model (MOD1) and adding random parameter $\xi_j$ as follows: 
107  <center>$ 
108  \min_{x\geq0}{\underbrace{\left\{\sum_{i=1}^nc_ix_i+\underbrace{\mathbb{E}_{\xi}\left[\min_{y\geq0}\left\{\sum_{i=1}^n \sum_{j=1}^kq_iT_jy_{ij}\right\}\right]}_{\text{second stage}}\right\}}_{\text{first stage}}}$</center><br> 
109  <ul><ul><ul><ul><em>subject to:</em><br></ul></ul></ul></ul> 
110  <center> 
111      <center> 
112  $ 
113  \begin{align} 
114  \text{first stage constraints:}&\\ 
115  &\sum_{i=1}^n x_i\geq \sum_{j=1}^k \xi_j^{max}\\ 
116  &\sum_{i=1}^nc_ix_i\leq c_{max}\\ 
117  \text{second stage constraints:}&\\ 
118  &\sum_{j=1}^ky_{ij}\leq x_i\quad \quad i=\{1,\cdots,n\}\\ 
119  &\sum_{i=1}^ny_{ij}\geq \xi_j\quad \quad j=\{1,\cdots,k\} 
120  \end{align} 
121  $ 
122  </center><br>
